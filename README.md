# kofa-word-format（Wordフォーマット自動作成）

Wordテンプレート（.docx）に対戦カード・チーム情報・スタジアム情報を差し込んで、
複数のWordを一括生成するWebアプリ。
公開URL: https://sagerbafcsec4.github.io/kofa-word-format/

すべてブラウザ内で処理し、ファイルは外部送信されません。

## 修正する人へ（重要）

- **画面とロジックは `index.html` の中に全部入っています**（唯一の正本）。
- 修正は **`index.html` を直して、このリポジトリにコミットするだけ**。数十秒で公開ページに反映。
- PWA（オフライン対応）なので更新後に古い表示が残ることがあります → **Ctrl+F5**（スマホは再読み込み）。
- 別デバイス・別のClaude/Coworkでも、**まずこの README と `index.html` を読めば概要を把握できます。**

> ⚠ 注意: `index.html` は約220KBで、**.docx を生成するライブラリ部分が圧縮（minify）されて埋め込まれています**。
> 文言や差し込み項目など軽微な修正は可能ですが、大きな作り替えは元のソース（あれば）から行う方が安全です。

## 何をするアプリか

1. ベースとなる **Word テンプレート（.docx・複数可）** を読み込む（テンプレ内にプレースホルダを記入）。
2. 対戦カード・シーズン・大会・節などを入力。チーム情報やスタジアム情報の .docx を読み込ませると、
   該当位置に**まるごと差し込み**（画像・表・書式そのまま）。
3. 試合ごとに Word を生成し、フォルダ保存またはZIP一括保存。

### 使えるプレースホルダ
- 文字差し込み: `{{HOME_TEAM}}` `{{AWAY_TEAM}}` `{{SEASON}}` `{{COMPETITION}}` `{{MATCHDAY}}`
- 丸ごと差し込み（位置の目印）:
  `{{HOME_INFO}}` `{{AWAY_INFO}}`（チーム情報.docxを差し込む位置）/
  `{{HOME_STADIUM_INFO}}` `{{AWAY_STADIUM_INFO}}`（スタジアム情報.docxを差し込む位置）

## 技術メモ

- 外部CDNライブラリは無し（.docx生成に必要なライブラリを index.html 内にバンドル）。
- チーム名→teamsシート→スタジアム名 で自動照合してスタジアム情報を差し込む。
